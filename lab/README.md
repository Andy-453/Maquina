# Laboratorio

Esta carpeta agrupa los servicios y artefactos que simulan a la victima dentro de la misma maquina:

- `vulnerable_apps/` para mini aplicaciones vulnerables.
- `fake_sites/` para dominios o sitios de practica.
- `hidden_services/` para servicios locales auxiliares.
- `files/` para binarios, imagenes o muestras forenses.

Separar esta capa del portal evita que el codigo base del CTF quede expuesto de forma accidental.
